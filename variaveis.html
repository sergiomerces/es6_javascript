Escopo de Variáveis

  • var: variável global
  • let: variável de bloco
  • const: variável de bloco

Escopo em JavaScript é quando digitamos algo dentro de {}. O mesmo vale para if, while, for ou escopo de function.

Declarando variáeis e atribuindo valores

var nameVar = 'Jobs';
let nameLet = 'Gates';
const nameConst = 'Torvalds';

console.log('nameVar: ${nameVar}');
console.log('nameLet: ${nameLet}');
console.log('nameConst: ${nameConst}');

var teste = 'Opa'; //escopo global

window.teste >> 'Opa'

O mesmo não aconte quando usamos let e const.

Usando apenas variavel global

var test = 'example';

(() => {
console.log('Valor dentro da função "${test}"');

if (true) {
var test = 'example';
console.log('Valor dentro do if "${test}"');
}

console.log('Valor após a execução do if "${test}"');
})();

com variável de bloco

(() => {
let test = 'example';
console.log('Valor dentro da função "${test}"');

if (true) {
let test = 'example';
console.log('Valor dentro do if "${test}"');
}

console.log('Valor após a execução do if "${test}"');
})();
